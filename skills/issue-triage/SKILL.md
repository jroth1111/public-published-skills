# Issue Triage

## Goal
Convert a report into a reproducible, bounded issue.

## Steps
1. Restate the issue in one sentence.
2. Capture environment, inputs, and expected vs actual behavior.
3. Define reproduction steps.
4. Define one verification path.
5. Classify severity and scope.

## Output Format
- Summary
- Repro steps
- Expected vs actual
- Verification
- Scope and priority
